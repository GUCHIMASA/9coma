# YouTube版 9TUBE：復旧・配色テーマ・シェア画像実装指示書 (Phase YT-4/5)

## 背景・目的
作業者AIの誤解により「キーワード検索」に変更されたロジックを、元々の仕様である「URLペースト方式（メタデータ自動取得）」に復元します。その上で、配色テーマ機能と高画質縦長シェア画像の実装を段階的に進めます。

> [!IMPORTANT]
> **このタスクは段階的に進めます。各フェーズの実装が完了するたびに、必ずユーザーに報告し、承認を得てから次のフェーズへ進んでください。**

## 1. 修正・新規作成対象
- `src/app/9tube/page.tsx` (検索ロジック復元・UI追加)
- `src/app/api/9tube/list/route.ts` (ID生成復旧・保存フィールド追加)
- `src/app/9tube/list/[id]/opengraph-image.tsx` (配色反映)
- `src/app/9tube/list/[id]/share-image/route.tsx` (新規：縦長画像生成)

---

## 2. 実装フェーズ

### フェーズ 1: コア機能の復旧と基盤整理
1. **検索ロジックの復元**: `page.tsx` を修正し、キーワード検索ではなく `YtSearchModal` を用いた「URLを貼るとプレビューが出る」方式に戻す。
2. **グリッドと操作性の復旧**: `YtGrid` を使用し、ドラッグ＆ドロップによるスロット入れ替え機能を復旧・実装する。
3. **ID生成の修正**: `api/9tube/list/route.ts` にて、`nanoid(10)` を使用したカスタムID生成と `setDoc` による保存ロジックを復元する。
4. **GETハンドラの追加**: `api/9tube/list/route.ts` に ID 指定でリストを取得する `GET` ハンドラを実装する。

> [!CAUTION]
> **ここで一旦止まり、既存のURLペーストで動画が追加でき、正しいIDで保存されることをユーザーに確認してください。**

### フェーズ 2: 配色・テーマ選択機能の実装
1. **カラーピッカー**: `page.tsx` に `COLOR_THEMES` に基づく角丸四角のカラーピッカーを実装。
2. **テーマ選択**: マスター定義（`THEMES`）に基づいた `themeId` 管理のプルダウン/リストを実装。
3. **UI/UX向上**: プレミアム感のあるデザインにブラッシュアップ。

> [!CAUTION]
> **ここで一旦止まり、作成画面の UI が理想通りかユーザーに確認してください。**

### フェーズ 3: OGP/縦長シェア画像の実装
1. **OGP反映**: `opengraph-image.tsx` に `colorThemeId` を反映し、背景色などを動的に変更。
2. **縦長シェア画像**: `share-image/route.tsx` を新規作成。1200x1500 サイズで、リッチな背景演出（サムネイルのぼかし等）を含む縦長画像を生成。

> [!CAUTION]
> **全実装完了後、share-image が正しく生成されることを確認し、最終報告を行ってください。**

## 3. 実装上の注意 (Satori/Yoga制約)
- `display: 'flex'` を親・子すべての `div` に明示。
- `borderRadius` は数値（px）で指定。
- `objectFit: 'cover'` は避け、コンテナサイズで調整。
